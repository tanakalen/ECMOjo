Provides the following functionality:
  • Framework which allows a program to run both as an Application and through Java Web Start. See Access* classes.
  • Provides file access to files which reside inside jars or are on the local file system. See Hookup* classes. Also gives access to images.
  • Allows to play audio files without worrying about audio formats and audio file management. See Audio*.
  • Allows to preload images for faster access during the execution of a program. See ImageLoader class.
