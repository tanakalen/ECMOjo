Readme

The project has the following directories:
  • build - Created during execution of ant. Contains library, application, webstart, obfuscate if the Ant core tasks are executed.
  • conf - Configuration files which will not be altered when the program is running.
  • data - Some data needed to create the software.
  • doc - Developer documentation.
  • lib - Library files of the project.
  • resource - Resources needed during the build of the software.
  • scenario - Scenario files, that might be altered during program execution.
  • src - Contains all the java source files.
  • web - Contains all the web project related web files.

The project's And build.xml file consists of the following tasks:

  • Core Tasks (not all tasks apply - depending on project):
    • application - Builds the application of the project.
    • webstart - Builds the Java Web Start of the project.
    • webapp-* - To deploy and undeploy a web application (.war).
    • library - Builds the library jar file of the project.
    • libconf - Builds the jar file for the conf directory.
    • obfuscate - Obfuscates and signs the library jar file of the project.
    • warfile - Builds the web application .war file to deploy in a web server.
  • Maintenance Tasks:
    • clean - Cleans project from all redundant files.
    • test - Executes all the test cases to check if the software works as required.
  • Documentation Tasks:
    • javadoc - Creates the javadoc of the software.
    • j2h - Creates the HTML version of the Java source files.
    • size - Returns the project size.
